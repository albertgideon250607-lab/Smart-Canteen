# Nandha Canteen - AI Smart Canteen System 🍽️🤖

Welcome to the **Nandha Canteen** project! This is an AI-powered smart ordering web application built to streamline the canteen ordering process. Customers can easily scan a QR code on their table, explore the digital menu, place their orders without waiting in line, and track them in real-time. 

## 🚀 Features

### For Customers:
- **Hassle-Free Login:** No passwords required! Simply drop in your Name, Phone Number, and Table Number.
- **Digital Menu & Cart:** Browse through various categories (Breakfast, Lunch, Snacks, Beverages), and seamlessly add multiple items to your cart.
- **Multiple Payment Options:** Choose between:
  - **GPay QR:** Scan the dynamic QR code directly.
  - **UPI:** Proceed via direct UPI link.
  - **Cash on Delivery (COD):** Pay directly when your food arrives at your table.
- **Live Order Tracking:** Get real-time updates as your order goes from 'Preparing' to 'Delivered'.

### For Admins:
- **Admin Authentication:** Secure admin login to manage canteen operations.
- **Live Order Management:** View incoming orders in real-time and dynamically update their status (e.g., Mark as Ready or Delivered) so that customers are notified instantly.
- **Dashboard & KPIs:** Gain insights with a graphical revenue chart (via Chart.js) and key performance indicators like Total Orders, Average Prep Time, and Total Earnings.

## 🎨 UI & Design
The application features a modern, premium **Dark Mode** aesthetic utilizing vibrant gradients, glassmorphism, smooth micro-animations, and clean typography (`Space Grotesk` & `Nunito` fonts) to deliver a truly engaging user experience.

## 🛠️ Technology Stack
- **Structure:** HTML5
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid, Keyframe Animations)
- **Logics & State:** Vanilla JavaScript (ES6)
- **Data Visualization:** Chart.js

## 📦 How to Use
1. Open up the `NANDHA_CANTEEN_COMPLETE_Final-2.html` file in any modern web browser.
2. Choose between **Customer Login** or **Admin Login**.
3. **Customer Flow:** Login with your table number -> Add food to your cart -> Checkout -> Complete Payment -> Track Order.
4. **Admin Flow:** Manage incoming orders and monitor business statistics visually.
